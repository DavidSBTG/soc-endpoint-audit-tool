# SOC Endpoint Audit Tool

## 🔍 Overview
PowerShell-based security audit tool for endpoint analysis.

## ⚙️ Features
- Event Log Analysis (4624, 4625, etc.)
- Suspicious Services Detection
- Network Connection Monitoring
- Persistence Detection (WMI, Tasks, Registry)
- Defender Status Check

## 🚀 Usage
```powershell
.\Invoke-SOCEndpointAudit.ps1 -Deep -VerboseMode
